# EDUC-Y645
This is the supporting repository for all course materials associated with Y645 (Covariance structure analysis). 

See the IU Canvas page for this course for 
  1) due dates;
  2) assignment submissions;
  3) current syllabus.

## Folder Structure
Each weekly folder contains some or all of the following:

- **slides/** — lecture slides (PDF)
- **code/** — R scripts for in-class examples and demonstrations
- **data/** — datasets used in class
- **handouts or other materials/** — supplementary notes, worked examples
